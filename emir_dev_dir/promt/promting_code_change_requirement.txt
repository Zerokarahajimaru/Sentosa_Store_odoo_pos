-  Logika Diskon Bundling/Paket Furnitur Otomatis

    Kaitan Studi Kasus: Toko dekorasi rumah sering menjual barang secara set (misal: Beli 1 Meja, diskon 20% untuk 4 Kursi). Ini juga mendukung "kenyamanan pembeli"

- Modifikasi: Menulis script Python pada model PoS yang otomatis membaca isi keranjang belanja. Jika kombinasi barang tertentu (Meja + Kursi) masuk bersamaan, sistem otomatis mengaplikasikan potongan harga tanpa kasir harus menghitung manual.
Nilai Plus Rubrik: Modifikasi murni pada alur logika sistem/perhitungan finansial, menunjukkan penguasaan kode ERP yang kuat.



-  Fitur "Unbundling" atau Pecah Barang Paket di Gudang

    Kaitan Studi Kasus: Manajemen stok produk yang tidak terorganisir

. Terkadang toko harus membongkar barang paket (misal: Set Piring isi 12) untuk dijual satuan.
Modifikasi: Menambahkan tombol kustom "Pecah Paket" di modul Inventory. Jika ditekan, stok barang "Set Piring" akan berkurang 1, namun otomatis menambah stok "Piring Satuan" sebanyak 12 buah secara otomatis tanpa harus input manual.
Nilai Plus Rubrik: Penyesuaian logika sistem yang memecahkan masalah kompleks terkait konversi satuan barang (Kriteria 7).
